美区 Apple ID 充值教程（支付宝/礼品卡）

2023-02-08

因为美区账户无法绑定国内发行信用卡，所以要买付费 App只能通过购买礼品卡的方式给美区账户充值然后才能购买付费 App。


目前美区可以用支付宝充值比较方便，所以准备俩个方案支付宝和礼品卡的充值教程。


支付宝充值流程

  • 进入到支付宝首页-顶部切换一个美国地址。
  • 底部导航键点击出境。
  • 看到“折扣礼卡”点击进入。
  • 下滑到最下面 “更多大牌折扣礼卡”点击进入。
  • 找到 “App Store & iTunes”
  • 填写个人信息，姓名，邮箱接收兑换码。
  • 输入需要充值的金额。
  • 点击立即购买即可。

礼品卡充值流程


Gift Card 是苹果开发的一种虚拟的充值卡。充值卡可以直接给 Apple ID 充值，用于购买 Apple Music，App Store 的软件等等。在苹果美国官网，这是最推荐的方法，安全、快速、简单，不用魔法上网，只要有外币信用卡就行。

注意事项：

  1. 地区一定要选择美国进行购买，在美国购买后不仅可以在APP Store进行消费，还可以在iTunes Store等苹果其他的商店进行消费。
  2. 要准备一张国内的双币卡，VISA 或者万事达 mastercard 的信用卡，没有信用卡是无法购买的。
  3. 邮箱，建议就用 Gmail 邮箱，如果用国内的邮箱有可能会收不到礼品卡的兑换码。

一、填写充值信息

  • 打开浏览器进入美国苹果官网：apple.com 地区选择美国。

  • 拉到最下面进入Gift Cards购买页，点击购买按钮Buy。

  • 邮箱，礼品卡的 LOGO随便选，输入想要充值的金额，最低只能10美元。

  • 收卡人姓名随意填、邮箱填写真实邮箱建议（Outlook / Gmail），用来接收礼品卡卡号的邮箱，切记慎重不能填错，确保邮箱可以接收邮件。

  • 发送者姓名随意填，发送者邮箱填自己国内邮箱。个性化信息也可以随意填。

  • 全部填写完后，点添加到 购物车 Add to Bag，然后点击检查 Check Out。


  • 接着会提示你登录你的 Apple ID，这里我们直接登录中国区的 Apple ID 就行了，后续查看发货进度，选择信任 “Trust” 。

二、付款信息填写


  • 选择第一个信用卡或借记卡 Credit or Debit Card。


信用卡信息：

从上到下依次是信用卡卡号、到期日期以及CVV。CVV 就是卡片背面后三位数。

账单地址：

地区这边不用改，默认的美国。填写你信用卡的名字，First Name 名，Last Name 姓。然后街道、X详细住址、邮编、城市、手机号。 美国地址生成器

  • 全部填写完成后，拉到最下面点击继续检查 Continue to Review。
  • 接着检查一下之前填写的邮箱等信息。

检查完成后，拉到底部，勾选条款，最后点 提交订单 Place Your Order。

到这里我们就已经下单成功了，页面中会出现“你的订单已提交”的字样 Your order has been placed”。

等待一段时间，之前填写的收件邮箱地址就会收到第一封邮件，提醒你“您的礼品卡订单正在处理中”。

接着再等待10分钟左右，会收到第二封邮件，第二封邮件中包含了礼品卡的卡号，收到卡号后，我们就能给美区的 Apple ID 充值了。

三、美区礼品卡充值教程

打开收到的第二封邮件（内含美区礼品卡兑换码），拷贝礼品卡卡号。

  1. 然后打开 App Store，记得先登录你的美区账号。
  2. 点击右上角的头像，然后点击 兑换礼品卡或代码 Redeem Gift Card or Code。
  3. 把卡号粘贴到 输入代码 Enter Code Manually 处，最后点击右上角的 兑换 Redeem。
  4. 充值过程中可能会提示你输入美区 Apple ID 的密码，按提示输入即可。
  5. 接着我们就可以看到，礼品卡已经充值成功，然后我们就可以去购买那些付费的 App 了。
