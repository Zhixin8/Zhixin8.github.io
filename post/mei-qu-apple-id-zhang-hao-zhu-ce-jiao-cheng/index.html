美区 Apple ID 帐号注册教程

2023-01-31

你是否真的需要一个美区 APPLE ID 账号？
或许对比国内账户已经很习惯了，但是世界那么大总是要看看的。
拥有美区账号，就可以下载 Instagram 关注喜欢的明星、YouTube 看到更多元化的视频、Netfilx Disney+ 更加优质的剧集影视 、 等等优质App。

第一步：访问 Apple ID 官网

1.1 在浏览器中打开此地址：https://appleid.apple.com/account

第二步：填写注册信息

2.1 填写美国身份信息 （美国地址生成器）

2.2 设置登录信息——点击继续

邮箱：输入邮箱地址 （建议使用国外账户）
密码：至少8个字符&大写与小写字母&至少一个数字
确认密码：再次输入设置的密码
手机区号：+86（中国大陆）
电话号码：输入您自己的手机号

第三步：验证邮箱和手机号

3.1 验证邮箱
打开注册邮箱输入收到的验证码

3.2 验证电话号码
输入收到的手机验证码

第四步：设置付款方式

4.1 添加付款方式
点击付款方式

点击添加付款方式

4.2 填写付款方式 （美国地址生成器）

勾选拷贝账单寄送地址

点击更新

第五步：在 App Store 登录

5.1 退出当前 Apple ID
在 iPhone 中打开 App Store——点击右上角头像——滑动页面至底部

点击退出登录

5.2 登录刚注册的美区 Apple ID
在 App Store 中点击右上角头像——输入 Apple ID 和 密码——点击 密码下方的登录——输入手机收验证——点击 检查



5.3 确认条款和支付方式

勾选「同意条款与条件」——点击下一页——点击下一页


5.4 恭喜 此 Apple ID 已经可用

当您看到此页面时即代表您的美区 Apple ID 已可正常使用。

现在可直接下载需要的的App

需要下载付费应用？

  • 优先建议直接官网购买礼品卡进行充值
  • 支付宝切换到美国购买——出境——折扣礼卡——最下面更多大牌折扣礼卡——App Store & iTunes US
